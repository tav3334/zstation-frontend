# ZSTATION - Gaming Station Management

Application de gestion de salle de gaming avec système de sessions, paiements et vente de produits.

## Fonctionnalités

- Gestion des machines gaming
- Système de sessions avec timer
- Paiements et génération de reçus
- Vente de produits (snacks, boissons)
- Tableau de bord administrateur avec statistiques
- Mode sombre/clair
- Interface responsive

## Technologies

- React 19
- Vite
- Lucide React (icônes)
- Axios pour les API
- CSS Modules

## Installation locale

```bash
npm install
npm run dev
```

## Backend

Cette application nécessite un backend Laravel. Assurez-vous que l'API backend est configurée et accessible.

### Configuration de l'API

Modifier l'URL de l'API dans `src/services/api.js`:

```javascript
baseURL: "https://votre-api-backend.com/api"
```
